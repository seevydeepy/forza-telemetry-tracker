# OKF Wiki Index

- [Desktop Backend](desktop-backend/wiki.html) - Python FastAPI desktop backend for local UDP telemetry capture, SQLite storage, lap/session analysis, export, replay, update checks, feedback submission, and FH6 world-map integration.
- [Web Dashboard](web-dashboard/wiki.html) - Svelte and Vite frontend embedded by the desktop app for live telemetry, replay, route review, settings, feedback, import, export, and dashboard visualisation.
- [Feedback Worker](feedback-worker/wiki.html) - Cloudflare Worker, D1 schema, and public feedback-reporting documentation for user-initiated anonymous reports into maintainer GitHub issue triage.
- [Release, CI, and Packaging](release-ci-packaging/wiki.html) - Windows installer packaging, release workflow, dependency locks, CI workflow, release documentation, licences, notices, and packaged-app smoke tooling.
- [FH6 Map Tile Converter](fh6-map-tile-converter/wiki.html) - .NET map tile conversion utility and vendored ForzaTechStudio readers used to generate local map cache assets from a user-selected FH6 install.
- [Project Docs and Community Metadata](project-docs/wiki.html) - Root project documentation, community metadata, support, privacy, security, funding, issue templates, design plans, and preview assets not owned by a narrower runtime or release bundle.
